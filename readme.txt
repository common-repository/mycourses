Add a course schedule to your Wordpress site and manage it.

== Description ==

This a very simple plugin that allows you to create a schedule of courses and manage them.

== Installation ==

1. Download mycourses.<version>.zip
2. Unzip the archive
3. Upload the 'mycourses' folder to your wp-content/plugins directory
4. Activate the plugin through the WordPress admin interface
5. Start adding sources through 'Tools' > 'MyCourses'
6. Customize your theme using the included sample-template.php as a guide

Currently, you must edit your theme to get the course schedule to show up. A short code
will be added in subsequent versions.
